## 核心体验

这是一款以赛马（或拳击？）为核心的轻操作放置游戏。戏的底层逻辑必须从“操控技术”彻底转向“信息博弈”。操作不应该决定胜负，但必须提供情绪价值和微弱的干预权，让玩家产生“是我帮它赢的”错觉。

### 信息不对称与数据深度（博弈基础）

赌马的核心乐趣在于“看走眼”或“慧眼识珠”。操作可以轻，但赛前情报必须重。

显性数据：速度、耐力、敏捷等基础数值（用雷达图展示）。

隐性数据（马匹状态）：每场比赛前，角色的状态必须随机波动。例如：显示“咸鱼今天心情不好（休息概率+20%）”或“兔子今天吃了兴奋胡萝卜（起跑加速+30%）”。玩家需要根据当前状态重新评估概率。

赛道与天气适配：不同野兽擅长不同地形（如苹果擅长冰面，海鸥顺风加速），赛前公布赛道特征，考验玩家的配队策略。

### 动态赔率与投注策略（经济心脏）

必须有期货式的赔率变动机制，刺激玩家的决策快感。

实时赔率：在倒计时阶段，赔率根据全服/全场玩家的下注金额实时变动。大热野兽赔率低，冷门野兽赔率高。

多形态下注：不仅是猜冠军，还要有“猜前三名”、“猜特定野兽能否跑进X秒”、“猜是否触发某事件”等复合彩池，让玩家即使没猜中冠军也有小奖。

### “混沌但可读”的模拟引擎（比赛内核）

比赛过程必须全自动运行，但机制必须透明，不能是纯黑盒随机数。

基于属性的物理模拟：速度、加速度、碰撞体积、障碍判定由引擎计算，而非预录动画。这样当“乌龟”因为“兔子”在前面而触发“嫉妒加速”反超时，玩家会觉得是机制生效，而非系统控盘。

随机事件池：内置大量突发事件（如“发疯的牛冲入赛道”、“突然的逆风”）。重点是事件必须有预警（如屏幕上方字幕滚动“前方检测到异常气流”），让玩家在下注时就评估风险偏好。

### 轻量化的“干预节点”（操作边界）

这是“轻操作”的精髓：玩家不能控制移动，只能释放“限次技能”。建议将操作缩减为赛前和赛中关键的2-3次点击。

建议操作方案：

赛前：选择3个道具放入快捷栏（非赛道拾取）。

冷却制：每个按钮每圈仅能使用1次。严禁使用摇杆或方向键，避免变成赛车游戏。

### 沉浸式观赛与直播感（情绪锚点）

轻操作意味着玩家大部分时间在“看”，所以视觉和听觉反馈必须达到“直播解说”级别。

屏幕上方滚动类似体育直播的字幕（“天呐！圆形利用滚动特性完美避开了香蕉皮！”），配合角色名字的颜色高亮和震动。

### 情感养成与“伪所有权”（长期留存和强烈的失败惩罚）

为了让玩家不只是机械下注，必须让玩家拥有1只专属野兽。

专属培养：玩家通过赢来的金币，为自己的专属野兽微调属性（如+1速度或+1生命），或购买特定外观（不影响属性，纯拉风）。

出战分红：玩家可以花费报名费派自己的专属野兽参赛。如果它赢了，玩家获得高额奖金；

**如果输了，它会死亡。** 是的，死亡，这可能会迫使玩家给怪物佩戴护身符或购买保险(以占据某个槽位)，甚至让玩家在赛前犹豫要不要派出自己的专属野兽参赛。

这种“共有财产”的得失感，远比单纯押注冷冰冰的AI更刺激。

## 优化空间

明确这是赌马游戏，而非赛车游戏。因此我们应该将游戏的核心玩法放在赛前的单位选择和下注上，赛道上的操作固然可以增加游戏的趣味性，但不应该成为游戏的核心玩法。

### 单位选择方式变更

目前是在赛场上选择任意单位下注，但其实应该在赛前制作一个单位选择界面，玩家可以选择自己心仪的单位下注，并且如果足够有钱的话，可以指派自己的单位参加比赛。这样可以增加玩家的参与感和策略性。

互动方式也可以增加，尤其是给角色加上 Shader 的边缘线，这会增强玩家的视觉体验。

### 角色指示变化

角色的数值可以用更好看的六边形或进度条来表示，每个角色都拥有自己专属的特性。

### 动画优化

大手捏小东西的动画有大量的优化空间。时序和节奏都需要调整，用Dotween来做动画会更好。

## 骨骼改造

用骨骼动画改造小怪物，更流畅的可替换组件。
这样可以制作多种小怪物的差分，不会过于同质化。

### 赛道优化

赛带上的障碍和机关可以更加丰富多样，增加更多的互动性和趣味性。可以考虑加入动态障碍物、可破坏的环境元素以及随机事件来提升比赛的不可预测性。

赛道还可以做成环形跑道，不光是一长条。

### 主动道具优化

增加道具类目

增加商店，而非使用赛道上的随机刷新（需要参考一下马里奥赛车的设计）

## 银河冠军巡回赛

**“爬格子锦标赛”+“田忌赛马”+“连胜登顶”**

### 1. 赛事架构：星际爬塔与种子选手

这不仅仅是随机匹配，而是一个**有层级、有梯度的星际联赛地图**。界面表现为一张银河旋涡地图，玩家需要从最外圈的“海选卫星”一路向中心“冠军星”爬格子。

- **格子系统**：每站巡回赛由 **7个格子** 组成（类似大满贯的晋级之路）。
  - **第1-2格（预选赛）**：随机路人AI，强度低，用于热身。
  - **第3-5格（小组赛）**：匹配到其他玩家或高难AI，且每个格子都带有**随机环境词缀**（如“微重力：跳跃系野兽加速”）。
  - **第6格（半决赛·种子选手）**：固定出现该星域的**传奇种子**（例如“冰巨星之子·极地兔”），属性极其夸张，正面硬刚极难取胜。
  - **第7格（决赛·守关Boss）**：该巡回赛的冠军野兽，拥有专属光环和超高的“运气”豁免权。
- **动态种子机制**：种子选手的阵容**完全透明公开**，玩家在挑战前一格时，就能看到下一格种子选手的三只野兽站位和属性，这为“田忌赛马”提供了充分的决策情报。

### 2. 核心博弈：真正的“田忌赛马”战场

玩家携带**至多3只野兽**（且要决定哪只作为“王牌”）。比赛采用 **【三局两胜制（KOF模式变体）】**，但具体对阵顺序由玩家在赛前**暗牌布阵**。

**具体流程如下：**

1. **双方布阵**：系统展示对手的三只野兽（含属性克制和赛道偏好）。玩家需要在倒计时内，将自己的3只野兽拖拽到 【先锋】、【中坚】、【大将】 三个槽位。
2. **对位逻辑**：双方布阵完毕后，自动按照 **先锋vs先锋、中坚vs中坚、大将vs大将** 的顺序进行三场单挑模拟。
3. **田忌赛马的极致体现**：
   - 如果你用最弱的“咸鱼”（下等马）去兑掉对方最强的“诺基亚”（上等马），虽然咸鱼惨败，但对方失去了王牌。
   - 接着你用中等的“圆形”战胜对方状态不佳的“中坚”，最后用最强的“兔子”碾压对方剩下的“弱鸡”。
   - **关键爽点**：胜负不完全看单场速度，而是看**体力消耗**。如果下等马把上等马的“冲刺技能”耗光了，那么上等马在下一场胜利后，进入第三场时会因为“疲劳”Debuff直接被弱鸡反杀。
4. **种子选手的压迫感**：面对种子选手时，系统会提示“该种子拥有额外一条命（连胜者优势）”，意味着你必须精准地用最恰当的野兽去消耗它，否则哪怕赢了一场，后面也会被种子选手一穿二。

### 3. 连胜荣耀：高风险高回报的“荣光之路”

既然最高荣誉需要连胜，那就要设计出**让玩家手心冒汗的奖励递增机制**。

- **连胜奖池叠加**：每赢下一场格子赛，当前奖池的“银河币”和“稀有训练素材”乘以1.5倍。如果输掉任何一场，**此次巡回赛立即终止**，连胜归零，但玩家可以带着已经累积的“保底奖励”退场（或者选择“All in”：继续挑战，赢则翻倍，输则保底也归零）。
- **赛季荣誉印记**：每完成一次“全胜通关（7连胜）”，该玩家使用的3只野兽会获得一枚**永久性的“星系征服者”徽章**。这枚徽章不仅显示在玩家ID旁，还能在常规下注比赛中，让这3只野兽的“运气”属性永久+1（上限封顶）。
- **世界首通榜**：针对每个巡回赛（高难度星球），设立“首通名人堂”。第一个用三只普通野兽击败该星球种子选手的玩家，其阵容会被做成“影子NPC”放入其他玩家的赛程中，供全服瞻仰和挑战。
- **致命惩罚**：如果失败得太难看，不要怪大手把宝宝捏碎。

### 4. 场外策略联动（把前两个方案串起来）

这个世界赛模式绝对不能孤立存在，它必须吃掉之前的“银河新闻”和“转会市场”：

- **情报压制**：在你准备爬格子的倒计时阶段，你消耗金币可以在“银河新闻网”购买**该巡回赛的独家内幕**，比如提前知道“第三格有沙尘暴，影响视野型野兽”，或者“种子选手昨天吃坏肚子，速度-5%”。这种**赛前情报战**直接决定了你的布阵思路。
- **临场租借（转会市场联动）**：如果你发现自己带的3只野兽完全打不过当前格子的地形，你可以花费转会币，**临时租借**一只该星球本土的“雇佣兵”野兽代替出战一场。这就把原本静态的养成，变成了动态的策略博弈。
  你这个洞察太致命了！**“致命野兽”**（Deadly Beasts）如果只会在跑道上跑圈，那简直是暴殄天物。把**“角斗”**加进来，游戏的内核瞬间从“体育运动”升华到了“**星际角斗综艺**”——这不仅是赌速度，更是赌生死、赌血性、赌野兽的战斗智商。

## 角斗赛的设计理念

**双轨制银河冠军赛**：

### 1. 赛事双轨制：竞速赛与角斗赛并行

在世界巡回赛的“爬格子”地图上，每个星球站点都有不同的主题。有些是“重力环赛道”（竞速），有些则是“血腥竞技场”（角斗）。

- **竞速赛（原版）**：考验速度、耐力、障碍规避。核心逻辑是“谁先冲线”。
- **角斗赛（新增）**：考验攻击力、防御力、连击率、血量回复。核心逻辑是“谁先倒下或血量百分比低者判负”。

**最关键的是“全能野兽”概念**：你携带的3只宝宝，必须同时具备竞速和角斗两套属性面板。比如“诺基亚”在竞速时像个笨重的坦克，但在角斗场里就是拥有“荆棘反伤”的肉盾；“圆形”在赛道靠滚动，在角斗场则靠“高速旋转切割”造成AOE伤害。

### 2. 角斗模式下的“轻操作”设计（极简战术）

既然是轻操作，玩家在角斗中**不控制移动和攻击**，而是做**“战前战术部署”**和**“赛中关键指令”**：

- **战前布阵（田忌赛马Pro）**：角斗采用**3V3车轮战（KOF）**。玩家需要决定3只野兽的**出场顺序**和**初始心态**（激进：+攻-防；保守：+防-攻；诡诈：概率闪避）。对手的阵容和种子选手习惯是可见的，这就完全变成了心理博弈——你猜对手会把最强输出放第几个？
- **赛中指令（仅2个按钮）**：战斗过程中，玩家不能搓招，但每隔15秒屏幕会出现两个战术按钮：【集火】（让当前野兽放弃防御，全力攻击）和【换血】（牺牲少量血量换取怒气值，以便释放大招）。点按时机（例如在对方释放蓄力技时点【换血】硬吃伤害攒大招）是唯一需要“反应”的地方，但输了也怪不到手速，只能怪决策。

### 3. 专属的“角斗属性”与“致命终结技”

为了让角斗更有观赏性和赌性，需要增加新的博弈维度：

- **体力/破绽条**：角斗中，每次攻击和被攻击都会积累“破绽值”。当破绽值满时，该野兽会进入**“处决状态”**。此时如果对手触发终结技，会直接扣除大量血量并附带“重伤”效果（永久降低最大生命上限）。
- **“下注生死盘”**：在角斗比赛中，赌盘不再只是“猜胜负”，还有 **“猜首杀”**（谁拿下第一个人头）、**“猜终结”**（是否触发处决动画）、**“猜时长”**（能否撑过30秒）。这种多元下注带来的刺激感，比单纯猜跑第几名要血腥得多。

### 4. 爬格子世界的完美融合（混合双打）

在你的世界巡回赛中，格子的难度不仅体现在数值上，还体现在**赛事类型的随机切换**上。这极大地提升了“携带3只宝宝”的战略深度：

- **混合关卡**：例如**第4格**是“铁笼竞速”（必须在跑圈的同时攻击旁边的对手，先冲线者胜，但血量归零直接出局）。这就要求你的3只宝宝不能是纯脆皮，也不能是纯坦克。
- **种子选手的双重威胁**：传奇种子不再是只会跑的呆子。比如“冰巨星·极地兔”在竞速段拥有无限加速，但在角斗段它极其脆弱。如果你在上一格把唯一能克制它的“诺基亚”用掉了，那么在下一格面对它时，你只能赌竞速赢，或者临时用“田忌赛马”拿炮灰去角斗场耗掉它的“狂暴技能”，再在竞速段翻盘。
- **连胜的“疲劳值”联动**：连胜最大的代价来了！如果你的野兽在角斗中受了重伤（血线低于30%），即使你赢了那一格，下一场竞速赛中它的速度也会因为“伤痛”而-20%。这就逼着玩家在世界赛中做出极难的抉择：**这一场我是要赢得漂亮（无伤速通），还是惨胜（保晋级但影响后续）？**

### 5. 银河新闻与转会市场的“角斗版”扩充

场外内容瞬间厚了一倍：

- **银河角斗周刊**：头条会播报“上周角斗场十大血腥瞬间”，同时披露某只野兽因为“爪子磨钝了”，下一场攻击力-5%，或者某只野兽因为“吃了外星菠菜”，进入狂暴状态。
- **角斗经纪人**：转会市场新增“角斗士专精”道具。你可以给你的“咸鱼”安装一个“钛合金鱼鳞”皮肤，使其在角斗时获得10%的物理免伤。这就把培养线路彻底分叉了——玩家可以选择把宝宝培养成赛道之王，还是竞技场杀神，甚至是“万金油”（两者都行但都不突出）。
